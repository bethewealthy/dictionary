# 00 · 제품 정의

> 이 문서가 다른 모든 결정의 상위 문서다. 기능을 넣을지 말지 다툴 때 여기 적힌 목표와 비목표로 판단한다.

## 한 문장

**초등학생이 어른 없이 혼자 열어도, 찾은 단어를 이해하고 · 소리 내어 읽고 · 며칠 뒤에도 기억하게 하는 영한사전.**

## 왜 만드는가

기존 선택지가 각각 실패하는 지점이 있다.

| 선택지 | 초등학생에게 실패하는 이유 |
|---|---|
| 네이버/파파고 사전 | 뜻이 나열식이고, 뜻풀이 자체가 표제어보다 어렵다. 예문이 성인 맥락 |
| Merriam-Webster Elementary | 품질은 최상이나 전부 영어. 한국 초등생 단독 사용 불가 |
| 단어 암기 앱 (퀴즐렛 등) | 단어-뜻 쌍만 있고 '사전'이 없다. 발음·예문·품사 정보가 빈약 |
| 종이 사전 | 찾는 데 시간이 걸려 아이가 포기한다. 소리가 없다 |

즉 **"사전의 정보 밀도"와 "아이가 혼자 감당할 수 있는 난이도"를 동시에 만족하는 물건이 없다.** 그 교차점이 이 프로젝트다.

## 대상

- **주 사용자**: 초등 3~6학년. 한글은 유창하게 읽고, 영어는 알파벳~기초 문장 단계
- **사용 맥락**: **혼자, 자율적으로.** 곁에서 설명해 줄 어른이 없다
- **규모**: 내 아이와 지인 몇 명. 불특정 다수 배포 아님

### 이 맥락에서 따라오는 설계 제약

혼자 쓴다는 조건이 가장 큰 제약이다.

1. **오답을 교정해 줄 사람이 없다** → 틀렸을 때 정답만 보여주면 안 되고, 소리와 예문을 즉시 함께 제시해 스스로 고치게 해야 한다
2. **막히면 물어볼 사람이 없다** → 화면에 설명이 필요한 UI는 실패다. 아이콘 대신 글자, 은유 대신 직설
3. **그만두게 할 사람도, 계속하게 할 사람도 없다** → 세션에 명확한 끝이 있어야 한다. 무한히 이어지는 화면을 만들지 않는다
4. **읽기 부담이 곧 이탈** → 한 화면에 한국어 지문이 3줄을 넘으면 다시 설계한다

## 해결할 문제와 우선순위

사용자가 지목한 네 가지 문제 전부를 목표로 하되, 아래 순서에 의존 관계가 있다.

| # | 문제 | 우선순위 | 왜 이 순서인가 |
|---|---|---|---|
| P1 | 기존 사전이 너무 어렵다 | **1순위** | 다른 셋의 토대. 뜻풀이가 어려우면 반복도 발음도 의미 없다 |
| P2 | 발음을 모른다 | **1순위** | P1과 함께 해결된다. 둘 다 '콘텐츠 품질' 문제라 앱 로직 없이 대부분 해결 |
| P3 | 외워도 금방 잊는다 | **2순위** | 좋은 콘텐츠 위에서만 간격반복이 값을 한다. 부실한 데이터를 반복시키면 잘못된 걸 굳힌다 |
| P4 | 뜻은 알지만 문장에 못 쓴다 | **3순위** | 가장 어렵고 가장 나중. P1~P3이 서지 않으면 시도할 수 없다 |

## 성공 기준

개인 프로젝트이므로 지표가 아니라 **관찰 가능한 행동**으로 정의한다.

- **M1**: 아이가 모르는 단어를 만났을 때, 다른 사전 대신 이걸 연다
- **M1**: 표제어 화면을 보고 어른에게 되묻지 않는다 (되묻는다면 그 표제어가 편집 실패)
- **M2**: 어른이 시키지 않아도 하루 학습을 스스로 시작한다
- **M2**: 2주 전에 학습한 단어를 다시 물었을 때 80% 이상 답한다
- **M3**: 배운 단어를 자기 문장에서 쓴다

## 비목표 — 하지 않을 것

명시적으로 만들지 않는다. 나중에 마음이 바뀌면 이 문서를 고치고 시작한다.

- **계정·로그인·서버.** 데이터는 기기 안에만 둔다. 아동 개인정보를 아예 수집하지 않는 것이 가장 안전한 설계다
- **수익화·광고·결제**
- **소셜 기능** (친구 랭킹, 공유, 대결). 혼자 쓰는 앱이고, 아동 대상 비교 경쟁은 의도적으로 배제한다
- **연속 학습일(스트릭) 압박.** 하루 빠졌다고 잃는 구조는 아이에게 부담으로 작동한다
- **전 학년·전 어휘 커버리지.** 어휘 수보다 표제어 하나의 품질이 우선이다
- **중고등 어휘, 시험 대비**
- **AI 챗봇 대화 상대**

## 제약

- **개발자 1인, 여가 시간.** 콘텐츠 입력이 개발보다 오래 걸린다는 걸 전제로 계획한다
- **저작권.** 기존 사전의 뜻풀이·예문을 옮기지 않는다. 전량 직접 작성한다
- **오프라인 동작.** 태블릿에서 네트워크 없이 완전히 동작해야 한다
