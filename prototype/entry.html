01 · 표제어 지면

학년을 바꾸면 지면이 바뀝니다

뜻갈래마다 난이도가 붙어 있어, 3학년에게는 기본 뜻만 보이고 6학년에게는 확장된 뜻까지 보입니다. 한 벌의 데이터로 세 학년을 감당하는 구조입니다.

과거형 ran · 과거분사 run · -ing형 running 불규칙

02 · 조판 해부

지면의 각 요소는 스키마 필드 하나입니다

영미 사전의 관습 중 초등 학습에 실제로 값을 하는 것만 남기고, 나머지(어원·인용문·이형 철자)는 잘라냈습니다.

syllables 음절 구분점

el·e·phant. 사전 지면의 정체성이자, 초등생이 긴 단어를 끊어 읽고 강세를 잡는 유일한 단서입니다. 별색으로 찍어 표제어보다 뒤로 물립니다.

ipa.us · audio.us 미국식이 기본

한국 초등 교과서가 미국식 기준이라, 학교에서 듣는 소리와 앱의 소리가 달라선 안 됩니다. UK는 water처럼 확연히 갈릴 때만 뒤에 병기합니다. 미국식 오디오는 필수 필드입니다.

재생 실패 경로 소리가 안 나도 학습은 끝난다

브라우저 정책·무음 모드·이어폰 미연결로 재생은 실제로 자주 실패합니다. 자동 재생 성공 여부와 무관하게 다시 듣기 버튼은 늘 같은 자리에 있고, 뜻과 예문은 전부 글자로도 읽힙니다.

senses[].pos 품사 색상 코드

이탤릭 약어(n. v. adj.)라는 사전 관습을 지키되, 아직 약어를 못 읽는 아이를 위해 품사별 색을 겹쳐 씁니다. 색만으로 구분하지 않도록 약어를 항상 함께 둡니다.

senses[].level 뜻갈래 난이도

run의 ‘달리다’는 3학년, ‘작동하다’는 5학년. 한 표제어를 학년별로 복제하지 않고 뜻갈래 단위로 노출을 제어합니다.

senses[].en 통제된 정의 어휘

Longman이 2,000단어로 모든 뜻을 풀듯, 정의문에 쓸 단어를 1,000개로 묶고 빌드할 때 자동 검증합니다. 이게 없으면 정의가 표제어보다 어려워집니다.

inflections.irregular 불규칙 굴절 표시

ran / run / running. 초등 단계에서 가장 자주 틀리는 지점이라 지면 위쪽, 뜻갈래보다 먼저 배치했습니다.
